    🎯 এই Repo সম্পর্কে¶

    এই CS Interview Handbook তৈরি করা হয়েছে বাংলাদেশের Junior Software Engineer পদপ্রার্থীদের জন্য। প্রতিটি টপিক আলাদা আলাদা ফাইলে সাজানো আছে — Fundamentals থেকে শুরু করে Advanced Concepts পর্যন্ত, Real-world Examples এবং Complete Interview Q&A সহ।

    লক্ষ্য: বাংলাদেশের শীর্ষ Software Company গুলোতে (Brain Station 23, BJIT, Reve Systems, Shajgoj, Pathao, Shohoz, Chaldal, Nagad, bKash, Therap) Technical Interview এ সফলভাবে উত্তীর্ণ হওয়া।

    💡 Note: সব content বাংলায় লেখা হয়েছে, Technical terms ইংরেজিতে রাখা হয়েছে — যাতে বোঝা সহজ হয় এবং Technical vocabulary-ও শেখা হয়।

    🗺️ Suggested Study Roadmap¶

    Junior SWE Interview এর জন্য নিচের ক্রমে পড়লে সবচেয়ে ভালো ফলাফল পাওয়া যাবে:

    ┌─────────────────────────────────────────────────────────────────┐
│                   🎯 Interview Preparation Path                 │
├──────┬──────────────────────────────────────────────────────────┤
│ Step │ টপিক                                                      │
├──────┼──────────────────────────────────────────────────────────┤
│  1️⃣  │ OOP — সবার আগে OOP মজবুত করুন (সব interview এ আসে)             │
│  2️⃣  │ DSA — Coding Round এর মূল ভিত্তি (12 PARTS সম্পন্ন ✅)            │
│  3️⃣  │ Database & SQL — প্রায় সব company তে SQL test হয়              │
│  4️⃣  │ Python — Backend scripting ও automation এর ভিত্তি            │
│  5️⃣  │ FastAPI — Modern Python API development                  │
│  6️⃣  │ JavaScript / React — Web-focused role এর জন্য              │
│  7️⃣  │ System Design — Mid-Senior level এ জিজ্ঞেস করা হয়             │
│  8️⃣  │ Git — Practical round এ কাজে আসে                          │
└──────┴──────────────────────────────────────────────────────────┘

    📖 হ্যান্ডবুক ব্যবহারের নিয়ম¶

    সেরা ফলাফলের জন্য নিচের পদ্ধতি অনুসরণ করুন:

    1. প্রথমে Theory পড়ুন — প্রতিটি concept এর Definition এবং Real-life Analogy বুঝুন
    2. Code Examples নিজে লিখুন — দেখে দেখে copy করবেন না, নিজের হাতে type করুন
    3. Interview Questions নিজে উত্তর দিন — আগে চেষ্টা করুন, তারপর model answer দেখুন
    4. Revision করুন — প্রতিটি Part শেষে Summary Table এবং Quick Q&A পড়ুন
    5. Mock Interview দিন — প্রতিটি file এ Mock Interview section আছে, সেটি practice করুন

    📁 File Naming Convention¶

    নতুন টপিক যোগ করার সময় নিচের pattern follow করুন:

    {TOPIC}_Junior_Engineer_BD_QA.md

Example:
  OOP_Junior_Engineer_BD_QA.md        ✅ (already exists)
  Database_Junior_Engineer_BD_QA.md   🔜
  JavaScript_Junior_Engineer_BD_QA.md ✅

    🤝 Contributing¶

    এই repo-তে আপনিও contribute করতে পারেন!

    • নতুন টপিকের file তৈরি করুন (naming convention অনুসরণ করুন)
    • Existing file এ নতুন interview questions যোগ করুন
    • Code examples এ bug fix করুন
    • Bengali writing improve করুন

    Pull Request এর আগে নিশ্চিত করুন যে content বাংলায় এবং Technical terms ইংরেজিতে লেখা আছে।
