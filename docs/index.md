# CS Interview Handbook

**বাংলাদেশের Software Engineer-দের জন্য সম্পূর্ণ Interview Preparation Guide**

> প্রতিটি টপিক বাংলায় সহজ ভাষায় ব্যাখ্যা করা হয়েছে। Code examples, real-world analogies এবং সম্পূর্ণ Interview Q&A সহ।

## এই Repo সম্পর্কে

এই **CS Interview Handbook** তৈরি করা হয়েছে বাংলাদেশের **Junior Software Engineer** পদপ্রার্থীদের জন্য। প্রতিটি টপিক আলাদা ফাইলে সাজানো আছে — Fundamentals থেকে শুরু করে Advanced Concepts পর্যন্ত।

**লক্ষ্য:** Brain Station 23, BJIT, Reve Systems, Shajgoj, Pathao, Shohoz, Chaldal, Nagad, bKash, Therap — বাংলাদেশের শীর্ষ Software Company-গুলোতে Technical Interview-এ সফলভাবে উত্তীর্ণ হওয়া।

## কীভাবে পড়বেন

1. **টপিক বেছে নিন** — বাম দিকের navigation থেকে হ্যান্ডবুক select করুন
2. **মনোযোগ দিয়ে পড়ুন** — প্রতিটি concept-এর explanation + code example পড়ুন
3. **নিজে লিখুন** — Code examples নিজে লিখে চালিয়ে দেখুন
4. **Interview Q&A** — প্রতিটি PART-এর শেষে Interview Q&A অবশ্যই পড়ুন
5. **Mock practice** — উত্তর না দেখে নিজে বলার চেষ্টা করুন, তারপর model answer মিলিয়ে নিন
